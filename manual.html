Pasos para crear el sitio web

  1. Se creó la estructura de carpetas y archivos HTML.
  2. Se usaron marcos (`frameset`) para dividir menú y contenido.
  3. Se diseñó un menú de navegación en `menu.html`.
  4. Se creó contenido HTML para cada sección (hoja de vida, notas, galería, etc.).
  5. Se insertó un video de YouTube usando un iframe.
  6. Se desarrolló un pequeño juego en JavaScript.
  7. Se probó todo el sitio en un navegador.
